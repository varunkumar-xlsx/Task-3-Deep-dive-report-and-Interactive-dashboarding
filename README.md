#   📊 US Retail Sales Dashboard — Power BI
 > A 4-year sales intelligence dashboard built wtih Power BI using the us Retail dataset(2015-2018).
 > Deep-dive segmentation analysis + interactive Power BI dashboard


## 🔍 Overview
- **9,800 transactions** across 4 years
- **4 US Regions** — East, West, Central, South
- **3 Categories** — Technology, Furniture, Office Supplies

## 📈 Key Insights
- 2018 was the best year — $722K revenue (↑ 20.3% YoY)
- West region leads all regions at $710K
- Technology is the top category at $827K (37.5% share)
- Total 4-year revenue: $2.26M

## 🛠️ Features
- Year-over-Year Sales, Orders & Customer Growth
- Interactive year slicer
- Monthly trend analysis
- Region & Category breakdown
- Executive Homepage

## 📐 DAX Measures Used
- Total Sales, Orders, Customers
- Sales LY / Orders LY / Customers LY
- YoY Growth % for all metrics
- Avg Order Value + AOV LY
